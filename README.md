# 🎮 2048 小游戏

一个用 Python 和 Pygame 编写的经典 2048 游戏，包含完整的用户系统和排行榜功能。

## ✨ 功能特点

- 🔢 经典的 2048 游戏玩法，使用方向键控制
- 👤 用户注册和登录系统
- 🏆 游戏排行榜，记录历史高分
- 📊 游戏记录查询，查看个人战绩
- 🎨 主题切换功能
- 💾 自动保存游戏进度

## 🖥️ 运行环境

- Python 3.7+
- Pygame 库
